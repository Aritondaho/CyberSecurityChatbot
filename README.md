# CyberSecurityChatbot

##  Description
This is a simple cybersecurity awareness chatbot built using C#.  
It helps users learn about online safety topics like passwords, phishing, malware, and scams.

---

##  Features
- Responds to user input using keywords
- Topics included:
  - Password safety 
  - Phishing 
  - Safe browsing 
  - Two-factor authentication 
  - Malware 
  - Scams 
- Friendly chatbot interaction
- Audio greeting feature

---

##  How to Run
1. Open the project in Visual Studio
2. Run the program
3. Enter your name
4. Ask about cybersecurity topics (e.g. "password", "phishing")
